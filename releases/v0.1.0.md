# mcp-registry-diff v0.1.0

Initial release of the MCP registry diff CLI.

## What changed

- Implemented registry normalization for list, `servers`/`items`/`repositories`/`mcpServers` lists, and object-keyed registries.
- Added diff detection for image, tag, command, env, auth, scope, network, and filesystem fields.
- Added Markdown and JSON reporting.
- Added `--format`, `--output`, and `--fail-on` modes.
- Added baseline tests and GitHub Actions CI.

## Try it

```bash
PYTHONPATH=src python3 -m mcp_registry_diff examples/registry-old.json examples/registry-new.json --format markdown
```
